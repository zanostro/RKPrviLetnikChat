Java 1.8 in novejše.

Program se zažene v ukazni vrstici ukazoma 
	java -jar server.jar 
	java -jar client.jar


Za pomoč uporabimo /help -- izpiše se nam seznam vseh ukazov

Privatna sporočila pošljemo z ukazom /private <username> <sporočilo>

Pomembno: Zaradi načina implementacije uporabniško ime ne sme imeti presledka v imenu. V nasprotnem primeru bo server prekinil povezavo.
